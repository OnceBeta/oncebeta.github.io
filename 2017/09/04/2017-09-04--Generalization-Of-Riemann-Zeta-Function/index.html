黎曼$\zeta$函数简括

发表于 2017-09-04 | 分类于 Math | | 阅读次数:

摘要

相信每一位数学爱好者对黎曼猜想(Riemann Hypothesis)都不会陌生，这是19世纪德国著名数学家黎曼于1859年在论文中提出的一个著名猜想。
该论文讨论了当时解析数论素数分布这一热门话题，黎曼在结尾处猜想道：黎曼$\zeta$函数的所有非平凡零点，都具有$s = \frac{1}{2} + it$的形式；换句话说，所有非平凡零点都在直线$\operatorname{Re} s = \frac{1}{2}$上。
这一猜想无数次在大小讲义或学术讨论会上被提到，却迄今悬而未决；单就其重要性而言，无论黎曼猜想被证明或被证伪，都将对数学界、物理学界及其他相关领域带来十分震撼的影响。
而很容易注意到，黎曼猜想研究的核心就是黎曼$\zeta$函数。这也正是本文将要介绍的。

背景介绍

调和级数$H = \sum\limits_{r = 1}^\infty {\frac{1}{r}} = 1 + \frac{1}{2} + \frac{1}{3} + \cdots$

这一和式几乎可被认为是无穷级数的起源了，早在中世纪就陆续有学者对其进行研究，但当时大多数学者认为这是一个收敛级数。直到1360年，数学家奥雷斯姆(Oresme)证明其发散，他的方法很简单：

将两个级数的项相对应，注意后一个级数$H’$，每一个括号内的项都小于前一个级数的对应项，即有$H’ < H$，而$H’$中每一个括号中所有项却都等于$\frac{1}{2}$，故$H’$趋于无穷，所以调和级数$H$也趋于无穷。

而在此之后，陆续有数学家对调和级数的发散速率也即渐近公式进行研究，但都以失败告终，直到牛顿(Newton)在他的《流数法》中提出了对数函数的泰勒级数展开式：

黎曼的论文

事实上欧拉还得到了其他一些零碎的结果，如：

但由于相对落后的分析基础，欧拉取得的这些结果都是不够全面的，我们并不否认欧拉在这一方面也有着不小的贡献，只是将欧拉所做的一切牢牢奠定在基石之上的人正是黎曼。

黎曼在这一方面所做的贡献主要有以下几点：

  1. 给出了$\zeta (s) = \sum\limits_{r = 1}^\infty {r^{ - s}}$的三种积分表示，即在复平面上解析开拓了$\zeta (s)$。
  2. $\zeta$函数正规化，即以$\zeta$函数为基础定义了满足函数方程$\xi (s) = \xi (1 - s)$的$\xi$函数。
  3. 证明了$\xi (t)$只有在$- \frac{1}{2} < \operatorname{Im} t < \frac{1}{2}$上才可能存在零点并给出了在带状区域$0 < \operatorname{Re} t < T$上零点数目的估计
(注：①这里变量记号$t$是黎曼论文原文中使用的记号，这是因为黎曼在论文中的定义与现在通常所见的定义不同，现代数学中的定义是由*朗道(Landau)*给出的： $$\xi (s) = \frac{1}{2}s(s - 1){\pi ^{ - \frac{s}{2}}}\Gamma (\frac{s}{2})\zeta (s)$$ 这里的自变量$s$与黎曼定义中的自变量$t$相差一个线性变换$s = \frac{1}{2} + it$，即将$t$以原点为中心逆时针旋转$90^\circ$并向右平移$\frac{1}{2}$个单位可得到$s$，而在朗道定义中，$\xi (t)$则被定义为大写的$\Xi (s)$。 ②类似地还有$\Pi (s)$，这是阶乘最初的解析开拓，即$\Pi (s) = \int\limits_0^{ + \infty } {e^{ - x}{x^s}dx} = s!$，它与现代数学中的$\Gamma$函数$\Gamma (s) = \int\limits_0^{ + \infty } {e^{ - x}{x^{s - 1}}dx} = (s - 1)!$间相差一个线性变换$s \leftrightarrow s - 1$。 为避免符号不统一，下文将统一采用现代数学的通用定义进行表述。)

历史背景

讲到黎曼猜想，就不得不提及猜想的出处——黎曼的长八页纸的论文了。
这是一篇具有划时代意义的论文，19世纪及以前，这一时期的数论主要研究堆叠数论方面的问题，关于素数分布的研究虽一直进行着，但自欧拉(Euler)提出$\sum\limits_{r = 1}^\infty {r^{ - s}} = \prod\limits_p {\frac{1}{1 - {p^{ - s}}}}$这一等式后，就一直未有显著的研究成果，1795年，天才数学家高斯(Gauss)提出素数定理的猜想，即：

当正整数$N$足够大时，有$\pi (N) \approx \int\limits_0^N {\frac{1}{\ln t}dt}$。
这一猜想在1896年被法国数学家阿达马(Hadamard)和比利时数学家普森(Poussin)先后独立地证明为更强的素数定理$\mathop {\lim }\limits_{x \to \infty } \frac{\pi (x)\ln x}{x} = 1$，该定理可以被认为是解析数论最杰出的成果之一，在后文还将介绍，因为它与黎曼$\zeta$函数边界值定理是等价的：

论文内容介绍

论文名为《论小于某给定值的素数的个数》，这是一篇长度仅有八页纸、内容却异常丰富的论文。其中包含了许多当时学界学界所不能理解的研究方法，其中黎曼惯用省略手法而著成的多处“显而易见”更是让同一时期他的同行们费尽心思，例如其中一处“显而易见”事实上就使用了47年后才被发现的梅林变换(Mellin Transformation)。
而所谓的黎曼猜想，不过是黎曼在论文结尾处简短而诙谐的一段结语：
“事实上我发现在该区间内的实根数目近似等于该数，极有可能所有根都是实数。对此我们自然希望能有一个严格的证明，不过在一些仓促的不成功的初步尝试之后，我暂把寻求证明搁在一边，因为对于我当前研究的直接目的来说他并不是必需的。”
